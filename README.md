# Convenience store server

> Providing api for convenience store

## Run with docker

Start dev enviroment:

```sh
docker compose -f docker-compose.dev.yaml up -d
```

Shutdown dev enviroment:

```sh
docker compose -f docker-compose.dev.yaml down
```
